# 画面遷移図

```
                    ┌─────────────────────────────────────────┐
                    │          左サイドバーナビゲーション          │
                    │  ホーム ／ 苦手問題一覧 ／ 科目・単元管理     │
                    └─────────────────────────────────────────┘

ホーム
（今日の復習一覧＋復習日未設定）
    │
    ├─ 問題をタップ ──→ 復習
    │                    │
    │                    ├─ 正解 ──→ 次の復習日を設定 ──→ ホームに戻る
    │                    │            ├─ 3連続正解なら「克服済みにしますか？」と提案
    │                    │            └─ 克服済みにする ──→ ホームに戻る
    │                    │
    │                    └─ 不正解 ──→ メモ・復習日を編集 ──→ ホームに戻る
    │
    └─ 「＋」ボタン ──→ 問題登録 ──→ ホームに戻る

苦手問題一覧
    │
    └─ 「復習する」ボタン ──→ 復習（同上）

克服済み一覧（苦手問題一覧内のタブで切り替え）
    │
    └─ 「苦手に戻す」ボタン ──→ active に戻す
```
