# 画面：科目・単元管理

**目的：** 科目・単元の追加・編集・削除を行う。

---

## 表示

- 科目一覧
  - 各科目の下に単元一覧を折りたたみ表示

---

## 操作

| 操作 | 詳細 |
|------|------|
| 科目を追加 | テキスト入力 → 保存 |
| 科目名を編集 | インライン編集 |
| 科目を削除 | 確認ダイアログ → 紐づく問題がある場合は 409 エラーメッセージ表示 |
| 単元を追加 | 科目ごとにテキスト入力 → 保存 |
| 単元名を編集 | インライン編集 |
| 単元を削除 | 確認ダイアログ → 紐づく問題がある場合は 409 エラーメッセージ表示 |

> 削除できない場合（409）は「先に問題を移動または削除してください」と案内する（[共通UI仕様](./common-ui.md) 参照）

---

## 空状態

| 条件 | 表示 |
|------|------|
| 科目が0件 | 「まず科目を追加しましょう（例: 数学・英語）」＋追加ボタン |

---

## 使用API

- `GET /subjects`
- `POST /subjects` / `PUT /subjects/{id}` / `DELETE /subjects/{id}`
- `GET /subjects/{subject_id}/units`
- `POST /subjects/{subject_id}/units` / `PUT /units/{id}` / `DELETE /units/{id}`
