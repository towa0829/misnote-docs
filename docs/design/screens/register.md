# 画面：問題登録

**目的：** 問題文・正解・間違えた理由・次の復習日を一画面で登録する。

---

## レイアウト

2カラム（左：問題・正解 / 右：メモ・復習日）

768px 未満では1カラムに積む（[共通UI仕様](./common-ui.md) 参照）。

---

## 入力項目

| 項目 | 入力形式 | 必須 |
|------|---------|------|
| 科目 | セレクトボックス | ✓ |
| 単元 | セレクトボックス（科目選択後に動的表示） | - |
| 問題文 | テキストエリア | ✓ |
| 正解 | テキストエリア | ✓ |
| 間違えた理由 | テキストエリア | - |
| 今回学んだこと | テキストエリア | - |
| 次の復習日 | 日付ピッカー | - |

---

## 操作

- 登録ボタン → 問題を保存してホームに戻る
  - メモまたは復習日が入力された場合、`mistake_notes` も自動作成（API側で処理）

---

## 使用API

- `GET /subjects`
- `GET /subjects/{subject_id}/units`
- `POST /questions`（`memo`・`learning`・`next_review_at` を同時送信）
